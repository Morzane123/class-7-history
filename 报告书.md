## 开发日志

### v1.6.0 - 图片视频压缩与视频上传

**新功能:**

1. 图片压缩功能
   - 使用 sharp 库进行图片压缩
   - 自动转换为 WebP 格式
   - 最大宽度限制 1920px
   - 压缩质量 80%
   - 头像压缩至 200x200px

2. 视频上传功能
   - 支持 MP4、WebM、MOV、AVI 格式
   - 单个视频最大 500MB（压缩前）
   - 创建 event_videos 数据库表存储视频信息
   - 事件创建页面添加视频上传组件

3. 视频压缩功能
   - 使用 fluent-ffmpeg 库进行视频压缩
   - H.264 编码，CRF 28
   - 最大分辨率 1920p
   - 压缩后目标大小 50MB
   - 自动调整码率确保大小符合要求

**新增依赖:**
- sharp - 高性能图片处理库
- fluent-ffmpeg - 视频处理库
- @types/fluent-ffmpeg - TypeScript 类型定义

**新增文件:**
- src/lib/imageCompress.ts - 图片压缩工具
- src/lib/videoCompress.ts - 视频压缩工具
- src/app/events/create/VideoUploader.tsx - 视频上传组件
- src/app/events/[id]/EventVideos.tsx - 视频显示组件
- src/app/api/events/videos/route.ts - 视频上传API
- src/app/api/events/[id]/videos/route.ts - 视频列表API

**数据库更新:**
- 新增 event_videos 表

**压缩效果:**
| 文件类型 | 原始大小 | 压缩后大小 | 节省空间 |
|----------|----------|------------|----------|
| 图片(5MB) | 5MB | 1-2MB | 60-80% |
| 视频(100MB) | 100MB | 30-50MB | 50-70% |

## 技术栈

- **前端**: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **后端**: Next.js API Routes
- **数据库**: SQLite (better-sqlite3)
- **认证**: JWT + bcryptjs
- **邮件**: nodemailer (腾讯企业邮箱)
- **图片处理**: sharp
- **视频处理**: fluent-ffmpeg
- **部署**: Ubuntu + PM2 + Nginx

## 技术决策记录

1. 选择SQLite数据库：轻量级、无需额外配置、适合小型项目
2. 选择Next.js App Router：现代化路由系统、支持服务端组件
3. 选择JWT认证：无状态、易于扩展
4. 选择Apple风格设计：简洁、现代、用户体验好
5. 选择毛玻璃效果：增强视觉层次感
6. 选择API提供文件服务：解决Next.js生产模式下的静态文件问题
7. 选择问题验证系统：确保七班用户身份真实性
8. 选择用户审核机制：增强系统安全性，防止恶意注册
9. 选择登录验证机制：保护班史内容隐私
10. 选择sharp进行图片压缩：高性能、支持WebP格式
11. 选择fluent-ffmpeg进行视频压缩：功能强大、支持多种格式
