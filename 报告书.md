## 技术栈

- **前端**: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **后端**: Next.js API Routes
- **数据库**: SQLite (better-sqlite3)
- **认证**: JWT + bcryptjs
- **邮件**: nodemailer (腾讯企业邮箱)
- **部署**: Ubuntu + PM2 + Nginx

## API接口

### 认证接口
- POST /api/auth/register - 注册
- POST /api/auth/login - 登录
- POST /api/auth/logout - 登出
- POST /api/auth/verify - 验证邮箱
- POST /api/auth/reset-password - 重置密码
- GET /api/auth/me - 获取当前用户

### 用户接口
- PUT /api/user/profile - 更新个人信息
- POST /api/user/avatar - 上传头像

### 板块接口
- GET /api/sections - 获取板块列表

### 事件接口
- GET /api/events - 获取事件列表
- GET /api/events/[id] - 获取事件详情
- POST /api/events/create - 创建事件
- PUT /api/events/[id] - 更新事件
- DELETE /api/events/[id] - 删除事件

### 评论接口
- GET /api/events/[id]/comments - 获取评论列表
- POST /api/events/[id]/comments - 创建评论
- DELETE /api/comments/[id] - 删除评论

### 管理接口
- GET /api/admin/users - 获取用户列表
- PUT /api/admin/users/[id]/role - 修改用户权限
- DELETE /api/admin/users/[id] - 删除用户
- POST /api/admin/sections - 创建板块
- PUT /api/admin/sections/[id] - 更新板块
- DELETE /api/admin/sections/[id] - 删除板块
- GET /api/admin/init - 初始化管理员账号

## 问题记录

1. PowerShell不支持`&&`语法，需使用`;`分隔命令
2. useSearchParams需要Suspense包裹
3. API路由需要标记为动态路由
4. 侧边栏状态需要提升到LayoutClient组件管理

## 技术决策记录

1. 选择SQLite数据库：轻量级、无需额外配置、适合小型项目
2. 选择Next.js App Router：现代化路由系统、支持服务端组件
3. 选择JWT认证：无状态、易于扩展
4. 选择Apple风格设计：简洁、现代、用户体验好
5. 选择毛玻璃效果：增强视觉层次感
