## 数据库模型

### 用户表 (users)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| email | TEXT | 邮箱 (唯一) |
| password | TEXT | 密码哈希 |
| nickname | TEXT | 昵称 |
| avatar | TEXT | 头像路径 |
| role | INTEGER | 权限等级 (1-3) |
| email_verified | INTEGER | 邮箱验证状态 |
| created_at | TEXT | 创建时间 |
| updated_at | TEXT | 更新时间 |

### 板块表 (sections)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| name | TEXT | 板块名称 |
| description | TEXT | 板块描述 |
| order | INTEGER | 排序 |
| created_at | TEXT | 创建时间 |

### 事件表 (events)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| section_id | TEXT | 所属板块 |
| title | TEXT | 事件名称 |
| content | TEXT | 事件内容 |
| event_date | TEXT | 事件发生时间 |
| author_id | TEXT | 作者ID |
| created_at | TEXT | 记录时间 |
| updated_at | TEXT | 更新时间 |

## API设计

### 认证接口
- POST /api/auth/register - 注册
- POST /api/auth/login - 登录
- POST /api/auth/logout - 登出
- POST /api/auth/verify-email - 验证邮箱
- POST /api/auth/reset-password - 重置密码

### 用户接口
- GET /api/user/profile - 获取个人信息
- PUT /api/user/profile - 更新个人信息
- POST /api/user/avatar - 上传头像

### 板块接口
- GET /api/sections - 获取板块列表
- POST /api/sections - 创建板块 (管理员)
- PUT /api/sections/:id - 更新板块 (管理员)
- DELETE /api/sections/:id - 删除板块 (管理员)

### 事件接口
- GET /api/events - 获取事件列表
- GET /api/events/:id - 获取事件详情
- POST /api/events - 创建事件
- PUT /api/events/:id - 更新事件
- DELETE /api/events/:id - 删除事件

### 管理接口
- GET /api/admin/users - 获取用户列表
- PUT /api/admin/users/:id/role - 修改用户权限
- DELETE /api/admin/users/:id - 删除用户
