# 璧山中学高2027届7班班史系统网站 - 任务书

## 项目概述

- **项目名称**: 璧山中学高2027届7班班史系统网站
- **开发团队**: 北域工作室
- **项目类型**: 班级历史记录Wiki系统
- **当前版本**: v1.7.0

## 技术栈

- **前端**: Next.js 14 (App Router) + TypeScript + Tailwind CSS
- **后端**: Next.js API Routes
- **数据库**: SQLite (better-sqlite3)
- **认证**: JWT + 邮箱验证
- **图片处理**: sharp
- **视频处理**: fluent-ffmpeg
- **部署**: Ubuntu + PM2 + Nginx

## 数据库模型

### 用户表 (users)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| email | TEXT | 邮箱 (唯一) |
| password | TEXT | 密码哈希 |
| nickname | TEXT | 昵称 |
| avatar | TEXT | 头像路径 |
| role | INTEGER | 权限等级 (0-3) |
| is_class7 | INTEGER | 是否七班用户 |
| class_name | TEXT | 班级名称 |
| approved | INTEGER | 审核状态 |
| email_verified | INTEGER | 邮箱验证状态 |
| created_at | TEXT | 创建时间 |
| updated_at | TEXT | 更新时间 |

### 板块表 (sections)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| name | TEXT | 板块名称 |
| description | TEXT | 板块描述 |
| sort_order | INTEGER | 排序 |
| created_at | TEXT | 创建时间 |

### 事件表 (events)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| section_id | TEXT | 所属板块 |
| title | TEXT | 事件名称 |
| content | TEXT | 事件内容 |
| event_date | TEXT | 事件发生时间 |
| author_id | TEXT | 作者ID |
| created_at | TEXT | 记录时间 |
| updated_at | TEXT | 更新时间 |

### 评论表 (comments)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| event_id | TEXT | 所属事件 |
| parent_id | TEXT | 父评论ID |
| content | TEXT | 评论内容 |
| author_id | TEXT | 作者ID |
| created_at | TEXT | 创建时间 |

### 事件图片表 (event_images)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| event_id | TEXT | 所属事件 |
| image_path | TEXT | 图片路径 |
| sort_order | INTEGER | 排序 |
| created_at | TEXT | 创建时间 |

### 事件视频表 (event_videos)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT | 主键 (UUID) |
| event_id | TEXT | 所属事件 |
| video_path | TEXT | 视频路径 |
| sort_order | INTEGER | 排序 |
| created_at | TEXT | 创建时间 |

## 数据库索引

- events: section_id, author_id, event_date
- comments: event_id, parent_id, author_id
- event_images: event_id
- event_videos: event_id

## API设计

### 认证接口

- POST /api/auth/register - 注册
- POST /api/auth/login - 登录
- POST /api/auth/logout - 登出
- POST /api/auth/verify - 验证邮箱
- POST /api/auth/reset-password - 重置密码
- GET /api/auth/me - 获取当前用户

### 用户接口

- PUT /api/user/profile - 更新个人信息
- POST /api/user/avatar - 上传头像

### 板块接口

- GET /api/sections - 获取板块列表

### 事件接口

- GET /api/events - 获取事件列表（分页）
- GET /api/events/[id] - 获取事件详情
- POST /api/events/create - 创建事件
- PUT /api/events/[id] - 更新事件
- DELETE /api/events/[id] - 删除事件
- POST /api/events/images - 上传图片
- DELETE /api/events/images/[id] - 删除图片
- POST /api/events/videos - 上传视频（支持SSE进度反馈）
- GET /api/events/videos/progress - 获取压缩进度
- DELETE /api/events/videos/[id] - 删除视频
- GET /api/events/[id]/videos - 获取视频列表

### 评论接口

- GET /api/events/[id]/comments - 获取评论列表（分页）
- POST /api/events/[id]/comments - 创建评论
- DELETE /api/comments/[id] - 删除评论

### 管理接口

- GET /api/admin/users - 获取用户列表
- PUT /api/admin/users/[id]/role - 修改用户权限
- DELETE /api/admin/users/[id] - 删除用户
- GET /api/admin/pending - 获取待审核用户
- POST /api/admin/pending/[id]/approve - 通过审核
- POST /api/admin/pending/[id]/reject - 拒绝审核
- POST /api/admin/sections - 创建板块
- PUT /api/admin/sections/[id] - 更新板块
- DELETE /api/admin/sections/[id] - 删除板块
- GET /api/admin/init - 初始化管理员账号
- GET /api/admin/migrate - 数据库迁移

### 问题接口

- GET /api/questions/random - 获取随机验证问题

### 文件接口

- GET /api/files/[...path] - 获取上传的文件（支持Range请求）
